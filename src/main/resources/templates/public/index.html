covid-19 lets help
Free, community based, exchange help to each other
This portal allows people to volunteer and help each other. During this Covid-19 pandemic situation, we see a lot of demand for effective coordination among people, citizens, institutions, NGOs, governments ...

Using this portal is very simple

  • One user can add all the items he needs after doing sign up
  • Another user adds all the items he has to offer as help
  • The two requirements are matched and both users are notified about it through email
